# hot-pot-timer

🍲 火锅计时器 - 精准掌握每种食材的最佳口感

一个专为火锅设计的计时器应用，帮助你完美掌控每种食材的涮煮时间。

## 功能特点

- 🥩 **分类管理** - 按肉类、蔬菜、海鲜等 6 大类别组织食材
- ⏱️ **多任务计时** - 支持多个食材同时计时
- 🔔 **智能提醒** - 计时结束时通过声音和震动提醒
- 📱 **移动优先** - 响应式设计，完美适配手机屏幕

## 食材类别

| 类别 | 图标 | 示例 |
|------|------|------|
| 肉类 | 🥩 | 羊肉片、牛肉片、牛肉丸 |
| 内脏 | 🫀 | 毛肚、鸭肠、鸭血 |
| 蔬菜 | 🥬 | 生菜、土豆、贡菜 |
| 海鲜 | 🦐 | 虾、鱼片、生蚝 |
| 豆制品 | 🧊 | 豆腐、豆皮、面筋 |
| 主食 | 🍜 | 面条、粉丝、玉米 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- React 19
- Vite 6
- 纯 CSS (无 UI 框架依赖)

## 使用场景

1. 打开手机浏览器访问应用
2. 选择食材类别（如"肉类"）
3. 点击食材卡片开始计时（如"羊肉片 15 秒"）
4. 继续选择其他食材，支持同时计时
5. 计时结束时会有声音和震动提醒

## 开发说明

### 添加新食材

编辑 `src/data.js`，在对应类别的 `items` 数组中添加：

```javascript
{ id: 'your-item', name: '食材名称', time: 秒数 }
```

### 修改推荐时间

直接调整 `src/data.js` 中对应食材的 `time` 值（单位：秒）

## 浏览器支持

- iOS Safari 12+
- Android Chrome 80+
- 其他现代浏览器

> 注意：震动提醒需要浏览器支持 Vibration API，声音提醒需要支持 AudioContext
