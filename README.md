# hot-pot-helper

🍲 火锅助手 - 你的火锅烹饪好帮手

一个多功能火锅辅助应用，包含**食材计时器**和**蘸料手册**两大核心功能。

## 功能特点

### ⏱️ 食材计时器
- 🥩 6 大类别、40+ 种常见火锅食材
- 🎯 每种食材都有推荐涮煮时间
- ⏲️ 支持多个食材同时计时
- 🔔 计时结束声音 + 震动双重提醒
- 🎨 完成状态视觉差异化显示

### 🥣 蘸料手册
- 📖 14+ 种经典蘸料配方
- 🗺️ 地方特色蘸料（东北、川渝、粤式、云南、贵州...）
- 🔍 搜索和收藏功能
- 🍲 锅底蘸料搭配推荐

## 食材类别

| 类别 | 图标 | 代表食材 |
|------|------|----------|
| 肉类 | 🥩 | 羊肉片、牛肉片、牛肉丸 |

## 使用场景

### 计时器
1. 打开手机浏览器访问应用
2. 选择食材类别（如"肉类"）
3. 点击食材卡片开始计时（如"羊肉片 15 秒"）
4. 继续选择其他食材，支持同时计时
5. 计时结束时会有声音和震动提醒，卡片变绿并显示✓标记

### 蘸料手册
1. 切换到"蘸料手册"页面
2. 浏览经典搭配、地方特色、特色配方
3. 点击蘸料卡片查看详细配方
4. 收藏常用配方，方便下次快速查找

## 开发说明

### 添加新食材

编辑 `src/data.js`，在对应类别的 `items` 数组中添加：

```javascript
{ id: 'your-item', name: '食材名称', time: 秒数 }
```

### 添加新蘸料

编辑 `src/sauces.js`，在对应类别的 `recipes` 数组中添加：

```javascript
{
  id: 'your-sauce',
  name: '蘸料名称',
  description: '描述',
  tags: ['标签 1', '标签 2'],
  ingredients: [
    { name: '配料名', amount: '用量' },
  ],
  tips: '小贴士',
}
```

### 版本号管理

```bash
# 手动递增版本号
npm run bump-version
```

版本号格式：`X.Y.Z`（主版本号。次版本号。修订号）
